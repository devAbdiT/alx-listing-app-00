# ALX Listing App

An Airbnb clone listing page built as part of the ALX curriculum.

## Description

This project is a web application that mimics Airbnb's property listing functionality. It displays property listings in an attractive, responsive layout with filtering and search capabilities.

## Project Structure

alx-listing-app/
├── components/ # Reusable UI components
│ └── PropertyCard/ # Displays individual property listings
├── interfaces/ # TypeScript type definitions
│ └── Property.ts # Defines the property data structure
├── constants/ # Application data and configurations
│ Contains available cities for filtering
├── public/assets/ # Static files and media
│ └── images/ # Property images and icons
└── README.md # Project documentation

### Directory Purposes

- **components/**: Contains reusable React components for building the UI
- **interfaces/**: Defines TypeScript interfaces for type safety and data structures
- **constants/**: Stores application constants, static data, and configuration values
- **public/assets/**: Holds static assets like images, fonts, and icons used throughout the application
